⚠️ Critical truth about PayFast

Most payment gateways (including PayFast-style systems) usually work like this:

❌ Default mode (most common)
Money goes to your merchant account first
You receive full payment
Then YOU manually/externally distribute commission

👉 So automatic split is NOT always built-in.

✅ Advanced mode (only if supported)

Some gateways support:

Marketplace payments
Split payments
Sub-merchant payouts

If PayFast supports split/marketplace payouts, then yes:
✔ commission can be auto deducted
✔ remainder sent to another account

BUT only if you are approved for that feature.

🧠 What YOU should implement (recommended architecture)

Even if PayFast does NOT support splitting, you can still build your system perfectly.

/payment-success

You handle:

verify transaction
mark booking = PAID
confirm booking automatically

transaction table
